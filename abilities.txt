Фишечки :)

ADHOC - удаленное управление клиентом [XEP-0050: Ad-Hoc Commands, XEP-0146: Remote Controlling Clients].
ANTISPAM - антиспам.
ARCHIVE - архив сообщений [почти то же, что и TEMPLATES].
AUTOSTATUS - автостатус (например, статус "away" через 5 минут и "extended away" через 10 бездействия).
AUTOTASK - планировщик задач (например, возможность автоматического выхода из конференции или клиента через определённое время).
BACK_IMAGE - возможность установить фоновую картинку для ростера и прочих окон.
CAPTCHA - поддержка капчи [XEP-0158: CAPTCHA Forms].
CHANGE_TRANSPORT - смена транспорта (перенос контактов с одного транспорта на другой).
CHECK_VERSION - проверка обновлений клиента.
CLIENTS_ICONS - иконки различных клиентов в ростере.
CLIPBOARD - буфер обмена.
COLOR_TUNE - возможность ручной настройки цветов в приложении.
CONSOLE - XML консоль.
DEBUG - отладочные возможности (вывод отладочной информации).
DETRANSLIT - функции транслитерации и детранслитерации.
ELF - эльфовые функции... траффик и данные для нижней панели.
FILE_IO - работа с ФС.
FILE_TRANSFER - передача файлов (SOCKS5 & IBB).
GRADIENT - возможность установить градиент в качестве цвета панелей окон (заголовок сверху и инфо-панель снизу).
HISTORY - сохранение истории сообщений.
HISTORY_READER - чтение истории сообщений в клиенте [см. также LAST_MESSAGES].
HTTPBIND - один из методов подключения, BOSH [XEP-0124: Bidirectional-streams Over Synchronous HTTP (BOSH)].
HTTPCONNECT - соединение через HTTP-прокси.
HTTPPOLL - использование метода HTTPPOLL для подключения к серверу [XEP-0025: Jabber HTTP Polling].
IMPORT_EXPORT - импорт/экспорт пользовательских данных (настройки, архив итп...).
JUICK - удобное использование бота juick@juick.com.
LANG_DEBUG - вывод сообщений о подгружаемых строчках перевода (на стандартный поток вывода).
LAST_MESSAGES - последние 5 входящих сообщений контакта автоматически загружаются из истории [см. также HISTORY_READER].
LOGROTATE - ограничивает размер списка сообщений контакта (диапазон 21...1000, настраивается).
MEMORY_USAGE - монитор памяти, когда-то был нужен для отладки расхода памяти
MIDP_TICKER - используется стандартный класс Ticker для "бегущей строки" (имеет ряд проблем на многих платформах)
NEW_SKIN - иконки visitors объединены с skin, иконки menu объединены с actions.
NICK_COLORS - раскрашивание ников.
NON_SASL_AUTH - использование pre-xmpp способа авторизации для серверов, сообщающих что они xmpp-compliant.
NOMMEDIA - не использовать MMAPI (JSR-135).
PEP - pubsub (поддержка в общем) + настроения (user mood) [XEP-0163: Personal Eventing Protocol, XEP-0060: Publish-Subscribe, XEP-0107: User Mood].
PEP_ACTIVITY - получение инфо состоянии контакта (требует PEP) [XEP-0108: User Activity].
PEP_TUNE - получение инфо о музыке играющей у контактов (требует PEP) [XEP-0118: User Tune].
PEP_LOCATION - получение инфо о местоположении контакта (требуется поддержка CLDC 1.1(JSR-179)) (требует PEP) [XEP-0080: User Location].
POPUPS - всплывающая панель [см. USE_ROTATOR].
PRIVACY - списки приватности [XEP-0016: Privacy Lists].
REQUEST_VOICE - пока не нужно... запрос голоса для визиторов.
RUNNING_MESSAGE - бегущая строка в заголовке окна набора сообщения.
SASL_XGOOGLETOKEN - механизм авторизации на серверах GoogleTalk.
SERVICE_DISCOVERY - диско (обзор сервисов) [XEP-0030: Service Discovery].
SE_LIGHT - подсветка на телефонах SonyEricsson и Nokia.
SYSTEM_NOTIFY - уведомления о новых сообщениях системными средствами телефона (Sony Ericsson).
SMILES - смайлы (эмоциконки).
STATS - статистика (использование памяти, свойства соединения).
TEMPLATES - шаблоны (возможность сохранять некоторые из сообщений) [почти то же, что и ARCHIVE].
TEST - временная абилити, для тестов (включать не рекомендуется).
TLS - поддержка STARTTLS.
USER_KEYS - настраиваемость комбинаций кнопок.
USE_ROTATOR - баллуны (всплывающие окна) и прокрутка длинных строк.
WMUC - без поддержки конференций.
XML_STREAM_DEBUG - вывод на стандартный вывод входящих и исходящих станз.
ZLIB - сжатие трафика.
