Фишечки :)

COLOR_TUNE - настройка цветов в приложении вручную
ARCHIVE - архив
POPUPS - всплывающая панель
ELF - эльфовые функции... траффик и данные для нижней панели
SERVICE_DISCOVERY - диско
PRIVACY - списки приватности
SMILES - смайлы
ANTISPAM - антиспам
REQUEST_VOICE - пока не нужно... запрос голоса для визиторов
HISTORY - история сообщений
HISTORY_READER - чтение истории сообщений в клиенте
SE_LIGHT - подсветка на сониэриках и нокиях
TEMPLATES - шаблоны
USER_KEYS - настройка комбинаций кнопок
AUTOSTATUS - автостатус
USE_ROTATOR - баллуны и прокрутка длинных строк
FILE_TRANSFER - передача файлов
LAST_MESSAGES - последние 5 входящих сообщений контакта, загружаются из истории
CHECK_VERSION	- проверка обновлений
DETRANSLIT - функции транслитерации и детранслитерации.
WMUC - без поддержки конференций
WSYSTEMGC - очистка кучи после интенсивного использования памяти
NICK_COLORS - раскрашивание ников
IMPORT_EXPORT - импорт/экспорт пользовательских данных
CHANGE_TRANSPORT - смена транспорта(перенос контактов с одного транспорта на другой)
CONSOLE - XML консоль
CLIPBOARD - Буфер обмена
GRADIENT - градиенты
PEP - pubsub(user mood, user tune, user activity)
PEP_TUNE - получение инфо о музыке играющей у контактов
PEP_ACTIVITY - получение инфо состоянии контакта(xep 0108)
CAPTCHA - капча
STATS - статистика
CLIENTS_ICONS - иконки клиентов
ADHOC - удаленное управление клиентом
LOGROTATE - задает размер списка сообщений контакта(диапазон 21...1000)
NEW_SKIN - иконки visitors объединены с skin, иконки menu объединены с actions
MENU_LISTENER - новый тип меню(заменяет родное)
RUNNING_MESSAGE - бегущая строка в заголовке окна набора сообщения
PLUGINS - обертка для плагинов
